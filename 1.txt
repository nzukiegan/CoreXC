1. Users

Stores system user accounts and authentication details.

| Column         | Type            | Constraints              | Description            |
| -------------- | --------------- | ------------------------ | ---------------------- |
| user__id       | INT IDENTITY PK | PK                       | Unique user identifier |
| username       | NVARCHAR(100)   | NOT NULL, UNIQUE         | Login name             |
| password__hash | NVARCHAR(255)   | NOT NULL                 | Hashed password        |
| full__name     | NVARCHAR(200)   | NOT NULL                 | User’s full name       |
| email          | NVARCHAR(255)   | UNIQUE                   | Email address          |
| is__active     | BIT             | DEFAULT 1                | Active status          |
| last__login    | DATETIME2       | NULL                     | Last login timestamp   |
| created__at    | DATETIME2       | DEFAULT SYSUTCDATETIME() | Record creation time   |
| updated__at    | DATETIME2       | DEFAULT SYSUTCDATETIME() | Last updated time      |
